# Game Engine 
## About
a python Game engine 

used for all my games 

## Controls 
Move: WASD

Aim: Mouse

Shoot Line: left ctrl

open console: F1
## Console Commands
static commands:
- clear -> clear history
- exit -> close console
- quit -> quit engine 
- kill -> kill the player

commands with 1 argument:
- gs (game_state_id) -> set scene in engine
